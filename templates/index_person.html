Detection Models

Primary Model: YOLOv8 Medium
Architecture: Convolutional Neural Network (CNN)
Classes: 80 COCO dataset classes
Speed: ~50-100ms per image
Accuracy: 53% mAP (multiple classes)
Size: ~50MB
Secondary Model: MobileNetV2
Architecture: Lightweight CNN with depthwise separable convolutions
Classes: Binary (Person / No Person)
Speed: ~10-20ms per image
Accuracy: 100% on person classification
Size: ~20MB
Ensemble Strategy
If YOLOv8 detects ≥1 person → Label: "PERSON" (99% confidence)
Else if MobileNetV2 probability > 0.5 → Label: "PERSON"
Else → Label: "NO PERSON"
